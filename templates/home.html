This is the Estudio Caribe back end

This will never be shown to users, and this page will dissapear once it's finished.

This two-part system prevents estudiocaribe vistors to access your database, protecting any sensitive data. It also handles all authentications and throttling of requests; for instance, no captcha is needed for the contact form, as this "server" will handle all security concerns. This back end will live securely in the same server as the front end, "server" its just a term for a different system.
